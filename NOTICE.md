# NOTICE

`t7kb.db` bundles knowledge from the Black Ops 3 modding community. Every row carries its `source` and `url`, so attribution travels with the data.

Sources:

- **gscode-api** — GSC/CSC built-in reference ([Blakintosh/gscode](https://github.com/Blakintosh/gscode))
- **source-scripts** — decompiled engine scripts ([shiversoftdev/t7-source](https://github.com/shiversoftdev/t7-source))
- **source-dump** — BO3 game-data dump ([ate47/bo3-source](https://github.com/ate47/bo3-source))
- **docs-bo3**, **defs-bo3** — Treyarch docs + mod-tools editor schema (shipped with the mod tools)
- **source-workspace** — the maintainer's workspace, with imported community packs credited inline
- **video-youtube** — tutorial transcript references, deep-linked to each video
- **personal-notes** — the maintainer's own notes
- **tools-dtzxporter** — tool READMEs
- **wiki-\*** — community wikis (each row links its source page)
- **forums-\***, **discord-bo3modtools** — community forum / Discord Q&A (anonymized)

The code in this repo is MIT — see [`LICENSE`](LICENSE).

**Removal:** if you authored something bundled here and want it taken out, message the maintainer [McReaper](https://github.com/McReaper) privately and it will be dropped from the next build.
