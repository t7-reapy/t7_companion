# Community

> **Status: stub.** The supporting cast. Wikis, Discords, contributors, asset packs. The community *is* the documentation for BO3 modding — knowledge is scattered across forums, Discord history, YouTube tutorials, and individual creators' websites.

## What this section will cover

- **Reference wikis** — where the canonical written knowledge lives.
- **Discord servers** — where the real-time / unindexed knowledge lives.
- **YouTube tutorial channels** — visual walkthroughs.
- **Contributors** — the asset-author registry: who made what, what their packs cover, where to find them.
- **Asset distribution chaos** — links rot. The README's dependency table exists as a lockfile because of this.

## Sub-pages (planned)

- `wikis.md` — T7Wiki, Modme, ardivee, zeroy, ZXG, UGX
- `discords.md` — MT, ZGC, IB, T7, X, BO3 modding servers
- `youtube.md` — tutorial channel index
- `contributors.md` — asset-author registry, what each one ships

## Reference reading (the major hubs)

- [T7Wiki](https://www.t7wiki.com/en/home) — best-maintained reference.
- [Modme Wiki — BO3](https://wiki.modme.co/wiki/Game-Support-_-Black-Ops-3.html) — older, deeper.
- [Modme Forums](https://forum.modme.co/wiki/forums/21.html) — searchable Q&A.
- [ardivee Wiki](https://wiki.ardivee.com/) — ambient rooms, tools (Acoustix, Cyph3r, LuiGUI).
- [zeroy Wiki](https://wiki.zeroy.com/) — GSC reference, ZM models, CoD WaW legacy that still applies.
- [ZXG (French)](https://www.zxg.fr/) — French modding community.
- [T7Mods](https://www.t7mods.com/) and [CODDB](https://coddb.net/) — asset repos / release hubs.
- [DEVRAW](https://www.devraw.net/) and the [DEVRAW Database Legacy spreadsheet](https://docs.google.com/spreadsheets/d/10aQLnuZUgvduFS4zgPNOTBlFbD-pBfpy--Gm9ilIqRg) — community asset registry.
- [UGX-Mods Forum](https://www.ugx-mods.com/forum/) and [UGX Confluence](https://confluence.ugx-mods.com/) — long-running WaW-era forum, fundamentals still apply.
- [IceGrenade BO3 Megathread](https://github.com/IceGrenade/bo3/wiki/Black-Ops-3-Mod-Tools-Get-Started) — orientation for newcomers.
- [Sharp's BO3 Modtools Assets forum](https://sg4yforums.com/Assets/BO3-Modtools-Assets/) — asset hosting.

## Status

This section is mostly a stable index — it doesn't change often once it's written, but the content drifts as Discord servers fragment / wikis go stale / contributors retire.
